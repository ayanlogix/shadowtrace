How ShadowTrace Works

ShadowTrace is a zero-knowledge identity protocol built to solve the "Privacy Paradox": how to prove who you are without revealing your data. By leveraging the Midnight Network's private state, we allow users to generate mathematical proofs of their attributes that can be verified on-chain with 100% certainty.

1

Secure Connect

Establish a cryptographically secure session by linking your Midnight-compatible wallet. This creates a temporary session key that manages the local ZK-environment.

2

Witness Gen

Your raw identity data (PII) is processed through a WASM-compiled Compact circuit directly in your browser's RAM. The raw data is instantly purged after use.

3

Anchoring

The final 128-byte proof is broadcasted to the Midnight Ledger. The network verifies the proof against the global state, anchoring your claim permanently.

Protocol Architecture

The ShadowTrace architecture utilizes a hybrid proof-of-work and zero-knowledge model anchored to the Midnight sidechain. Our architecture ensures that 100% of PII stays on the client device.

Developer SDK

Integrate ShadowTrace ZK-Identity into your own Web3 DApp in just 3 lines of code.

TypeScript — v1.4.2 Copy
// Import the ShadowTrace Prover
import { ShadowTraceProver } from '@shadowtrace/sdk';

// Initialize Prover with User Wallet
const prover = new ShadowTraceProver(walletConnector);

// Request ZK Proof for Age >= 18
const proof = await prover.verifyConstraint('AGE_OVER_18');

console.log(proof.hash);
